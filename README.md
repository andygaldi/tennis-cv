# Description
Perform the following steps to run:
```
# Create and activate virtual env
pyenv install 3.10.4
pyenv virtualenv 3.10.4 tennis-cv
pyenv activate tennis-cv

# Install requirements
poetry install

# Run code
python 2-determine-joints.py
```
